# Project Documentation Index

This directory contains the core strategic, structural, and operational documents for the **Jabi Lake AI-ESG Triathlon** and the emerging **AI-Agent Workforce Market**.

## Overview

The project is designed as a live proof-of-efficiency environment where a structured AI-agent workforce is used to organize, monitor, and govern a triathlon event at **Jabi Lake**. The system combines:

- real-time venue intelligence
- AI-agent orchestration
- ESG-aware operations
- field execution support
- reporting and governance logic
- and the future foundation of an agent marketplace/workforce core

At the center of the hierarchy is **Tarkwa**, the Guardian AI-Agent responsible for preserving mission coherence and coordinating all subordinate agents.

## Current architecture focus

The current documentation establishes the first strategic and governance layer for the repository. The next build focus is the **agent marketplace/workforce core**, including:

- agent identity and capability models
- task allocation and matching logic
- performance scoring and reputation logic
- workforce orchestration rules
- operational state tracking
- and reusable deployment structure for future events
