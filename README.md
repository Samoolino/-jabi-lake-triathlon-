## Jabi Lake AI-ESG Triathlon

The **Jabi Lake AI-ESG Triathlon** is a live proof-of-efficiency initiative designed to demonstrate how an **AI-Agent Workforce Market** can coordinate a full triathlon event using structured AI-agent roles, real-time venue data, and ESG-aligned execution.

At the center of this system is **Tarkwa**, the Guardian AI-Agent responsible for orchestrating all workforce agents across planning, sensing, reporting, logistics, and governance.

## Core Focus

- AI-agent workforce orchestration
- live venue intelligence (air, moisture, access, safety)
- ESG-driven event monitoring
- structured operational hierarchy
- marketplace-based workforce model
